## Purpose

Human wants to move toward a true 3D formation as soon as possible.

The current branch cannot honestly claim that yet.

So this note defines the minimum honest path from:

- soft planar morphology ownership

to:

- true 3D formation ownership

without pretending that full `XYZ = 4 / 5 / 4` runtime ownership already exists now.

After the current PR #6 carrier work, the maintained path can honestly claim:

- `X-forward` planar reference ownership
- `Y` lateral morphology participation in the reference surface
- stable expected/reference spacing semantics
- stable physical minimum spacing semantics
- a real restore line

That is the current honest owned meaning.

What it cannot yet honestly claim:

- runtime-owned `Z` morphology target

- full `XYZ = 4 / 5 / 4` runtime execution meaning

### Honest read not yet available

Current branch cannot yet support:

Planar-owned now means:

This is already meaningful progress.

## Minimum Next Ownership Opening for True 3D

The next honest opening should not jump straight to:

- full 3D slot surface

The better next opening is likely:

1. **3-axis morphology target**
   - forward extent
   - lateral extent
   - vertical extent

2. **Z-band ownership**
   - upper / middle / lower
   - or an equally bounded vertical grouping read

3. **3D preferred placement bands**
   - not exact volumetric slot lock
   - still morphology-first

This is preferable to reopening exact 3D slots too early.

## First Honest 3D Formation Read

The first honest 3D formation read should probably mean:

- forward axis is owned
- lateral extent is owned
- vertical extent is owned
- broad vertical band membership is owned
- compaction is continuous and morphology-preserving in all active axes

It should not yet mean:

- perfect volumetric template lock
- every-unit exact 3D target point
- every-tick global optimal 3D remap

## Why Soft Morphology Must Stay Central

The current branch history already shows the danger:

- rigid slot maps become hard-lock reads too easily

So future 3D ownership should not be opened as:

- a bigger rigid template

It should be opened as:

- larger-axis morphology ownership
- bounded band/region ownership
- continuous compaction ownership

That is the only honest path to "real 3D formation" without re-creating rigid lock in another dimension.

## Relationship to Turning-Cost

Future true 3D ownership and future turning-cost are related.

If heading realization stays too close to instant:

- even a good 3D morphology carrier may still look over-clean or snapping

So the best long path is:

1. clarify formation ownership honestly
2. settle soft planar morphology enough to trust the substrate
3. then later add turning-cost / heading-inertia as movement-side modernization

That ordering preserves semantic clarity.

## Bottom Line

The minimum honest path from PR #6 to real 3D formation is:

- soft planar morphology first
- then 3-axis morphology target
- then bounded Z-band ownership

Not:

- immediate full 3D slot ownership

Current branch can honestly say:

- first credible soft planar formation substrate candidate

It cannot honestly say:

- solved true 3D formation runtime ownership

That distinction should stay explicit in future governance and engineering discussion.
