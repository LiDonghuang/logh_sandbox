# Step 3 3D PR #6 Round 0 Formation-Transition Carrier Redefinition Note

Status: round-0 carrier redefinition note  
Scope: PR `#6` formation-substrate main line  
Authority: engineering redefinition note for governance review preparation only  
Non-scope: merge request, default switch, runtime-core rewrite, hostile-contact redesign

---

## I. Purpose

This note records the next carrier redefinition on top of the existing PR `#6` learning.

PR `#6` has already established bounded but real assets:

- expected/reference spacing legitimacy
- physical minimum spacing split legitimacy
- restore-line legitimacy
- maintained battle-path visibility of the restore line
- human-readable artifact discipline
- bounded failure learning about rigid reference-only and over-active banded carriers

The present purpose is **not** to discard that work.

The present purpose is to redefine the next carrier more honestly.

---

## II. What Is Now Rejected

The next carrier should no longer be framed as:

1. define a cleaner reference surface
2. let runtime follow it

That framing is now rejected as the main carrier read.

Why:

- matched-shape cases (`1 -> 1`, `4 -> 4`) repeatedly drift toward plate-like or over-clean motion
- mismatched-shape cases (`1 -> 4`, `4 -> 1`) fail the actual formation requirement
- human-readable motion exposed failures that cleaner local metrics did not properly penalize

So the current problem can no longer be treated as:

- a better slot map problem
- a better band-center problem
- a better restore-strength problem

---

## III. New Governing Read

The next carrier should be read as the smallest honest **formation-transition carrier**.

That means the carrier must explicitly own three coupled layers:

### 1. Target morphology

What morphology the fleet is trying to become.

Minimum current read:

- `morphology_axis`
- `extent_state`
  - `forward_extent`
  - `lateral_extent`
- `center_wing_differential`

### 2. Transport / topology continuity

How units or substructures move continuously from current shape toward target shape.

Required posture:

- continuous
- topology-aware
- bounded
- anti-rupture
- anti-shell-success-with-chaotic-interior

### 3. Movement realization

How turning, slowing, arrival, and hold/await actually realize that transition in motion.

Required posture:

- bounded
- readable
- human-credible in animation
- not merely locally metric-clean

---

## IV. Smallest Honest First Carrier

The smallest honest first carrier on top of PR `#6` learning is:

> keep the spacing split and restore-line legitimacy that PR `#6` already established, but stop treating formation as a better reference map; instead, treat formation as a continuous transition problem jointly owned by target morphology, topology-preserving transport, and bounded movement realization.

This read keeps what was valuable in PR `#6`, while explicitly rejecting the false hope that reference-surface cleanup alone can solve the formation problem.

---

## V. What Remains Useful From PR `#6`

The following remain valid and useful:

- spacing split as a real semantic distinction
- restore line as a real mechanism line
- battle-path visibility of that line
- band legitimacy as topology membership
- failure evidence showing that band actuation must not become explicit subformation ownership

So PR `#6` should not be read as disposable.

It should be read as:

- a preserved bounded learning branch
- whose next carrier must be re-carried, not abandoned

---

## VI. Immediate Boundary For The Next Round

The next round should not begin from:

- more scalar restore tuning
- more slot/band surface tuning in isolation
- hostile-contact repair as the main line

The next round should begin from:

- continuous morphology definition first
- then transport/topology continuity
- then the smallest turning/slowing/hold seam needed for plausibility

Bands may remain as topology membership.

Bands must not remain the first geometry-owning object.

---

## VII. Bottom Line

PR `#6` should now be carried forward under a new read:

**not reference-only formation restoration, but first bounded formation-transition ownership.**

That is the round-0 redefinition.
