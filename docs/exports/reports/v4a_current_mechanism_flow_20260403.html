0. Reading Guide

这份导出只描述当前 v4a 机制是怎样流动的：

  • 哪些层当前在拥有行为
  • 信号如何在 battle / formation / combat / terminal 之间传递
  • 当前 Human review 主要观察面是什么

颜色编码使用当前五类 discontinuity 分类，但这里仅作为 分类图例，不是好坏评价。

Class A: Owner / State
Class B: Reference / Axis
Class C: Force / Speed-Envelope
Class D: Targeting / Combat
Class E: Terminal / Settle
Neutral gray: shared plumbing / handoff

Arrow Legend

  • -->|stage / calls|: stage progression or same-tick function call order
  • -->|write now / direct feed|: direct state write or direct current-stage data feed
  • -.->|store / bias later|: computed now, stored in bundle/state, consumed later by another stage

  • fleet-centroid trajectory
  • alive-unit body observation
  • 3D HUD battle-focused rows:
    • Kinetics: v / h
    • eng_act
    • front_rw
    • fire_da

这些是观察面，不是 runtime owner。
