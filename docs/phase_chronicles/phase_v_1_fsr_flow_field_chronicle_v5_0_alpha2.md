## I. Background

During Phase V development, a structural inconsistency was observed:

-   As fleet size decreased due to attrition,
-   Formation outer scale did not contract proportionally,
-   A visible geometric gap emerged between inner and outer shells.

This was classified as a geometric-scale inconsistency rather than a
tactical or combat-layer issue.

------------------------------------------------------------------------

## II. Formation Scale Relaxation (FSR)

FSR introduced adaptive formation scaling:

R_eq(N) = R_eq(N0) \* sqrt(N / N0)

Design constraints: - FR modulates contraction speed only - Combat logic
unchanged - Targeting unchanged - Deterministic preserved

### Result

-   Outer-shell contraction restored
-   Visual coherence improved
-   Mirror metric improved
-   Determinism preserved

However, persistent outliers emerged in high-FR engagements.

------------------------------------------------------------------------
